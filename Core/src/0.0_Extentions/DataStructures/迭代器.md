# 迭代器

## 分类
- 输入迭代器 ``Input_Iteraror``
- 输出迭代器 ``Output_Iterator``
- 前向迭代器 ``Forward_Iterator``
- 双向迭代器 ``Bidirectional_Iterator``
- 随机访问迭代器 ``Random_Access_Iterator``

## 通用接口

### 类型定义
- ``value_type``: 元素类型
- ``reference``: 对应的引用类型
- ``pointer``: 对应的指针类型
- ``iterator_category``: 迭代器类别
- ``difference_type``: 元素之间的距离类型

### 函数接口

- ``operator*()``: 返回当前元素的引用或值
- ``operator==``: 比较两个迭代器是否相等
- ``operator!=``: 比较两个迭代器是否不相等
- ``operator++()``: 将迭代器移到下一个元素，返回迭代器本身。
- ``operator++(int)``: 返回当前迭代器，然后移动到下一个元素。

## 独有接口
- 输入迭代器:

    - 无（输入迭代器的特有接口主要是其只支持单向遍历的特性）

- 输出迭代器:

    - ``operator=``: 将值赋给当前元素

- 前向迭代器:
  
    - 支持拷贝构造和拷贝赋值。

- 双向迭代器（Bidirectional Iterator**

    - ``operator--()``: 将迭代器移到前一个元素

    - ``operator--(int)``: 返回当前迭代器，然后移动到前一个元素

- 随机访问迭代器（Random Access Iterator）

    - ``operator+()``: 返回一个新的迭代器，指向当前迭代器之后指定距离的位置

    - ``operator-()``: 返回一个新的迭代器，指向当前迭代器之前指定距离的位置

    - ``operator+=()``: 改变当前迭代器的位置
  
    - ``operator-=``: 改变当前迭代器的位置
  
    - ``operator[]``: 返回当前迭代器偏移指定距离后的元素
  
    - ``operator<、operator<=、operator>、operator>=``: 用于比较两个迭代器的相对位置

- 散列存储容器
